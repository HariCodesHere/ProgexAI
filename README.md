## ğŸ“¦ Getting Started

### Prerequisites
- Node.js & npm
- Python 3.x
- MongoDB
- GitHub Account
- OpenAI API Key or LLM model (if local)

### Installation

```bash
# Clone the repo
git clone https://github.com/HariCodesHere/ProgexAI.git
cd ProgexAI

# Install frontend & backend
cd client && npm install
cd ../server && npm install

# Start the backend
npm run dev

# Start the frontend (in another terminal)
cd ../client && npm start
```

### GenAI Engine (Python)

